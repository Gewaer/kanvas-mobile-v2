### Folder structure

we use Domain Driven pattern [info](https://medium.com/@hassan.djirdeh/domain-driven-react-redux-a474ecf7d126) with some little changes.

```
  - src
    - assets
    - config
    - components
      - block
      - elements
      - modifers
    - styles
      - colors
      - helpers
      - fonts
      - images
    - utils
    - modules
      - Folder
        - components
          -file
        - styles
        - action
        - index
        - reducers
        - types
        - sagas
        - test
        - @folder


```
